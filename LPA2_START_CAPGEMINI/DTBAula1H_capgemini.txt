### Exemplo prático:
Se você quiser atualizar a quantidade em estoque para **50** no produto com `id = 1`, use:
```sql
UPDATE Produtos
SET qtdeEstoque = 50
WHERE id = 1;
```
**Importante:** sempre tenha cuidado ao usar `UPDATE`, especialmente sem o `WHERE`, pois isso atualizaria *todas* as linhas da tabela.

SELECT COUNT(P.ID), C.NOME
FROM PRODUTOS P INNER JOIN CATEGORIAS C
ON P.CATEGORIAID = C.ID
GROUPBY P.CATEGORIAID
